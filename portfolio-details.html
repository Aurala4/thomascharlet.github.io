Description du projet (SAE 1.04)

Ce projet académique consiste à créer un site web personnel (portfolio) en respectant un cahier des charges précis. L'objectif est de développer une page de présentation, un CV en HTML/CSS, et un portfolio présentant les 5 compétences du BUT R&T.

Fonctionnalités Clés

Adaptation de Template Bootstrap

Utilisation du template SnapFolio et adaptation de sa structure (menu, sections, pied de page) pour correspondre aux besoins du projet.

Responsive Design

Le site est conçu pour s'adapter à 3 formats d'écrans (mobile, tablette, ordinateur) en utilisant les grilles Bootstrap.

Hébergement GitHub Pages

Le site est déployé via Git sur GitHub Pages, en respectant la nomenclature `.github.io` et en utilisant des chemins relatifs.

CV en HTML/CSS

Le CV n'est pas un PDF, mais est entièrement codé en HTML et stylisé en CSS pour une meilleure intégration (Section `#resume`).
